## 设计哲学

> 解决"为了几个小功能装一堆 App"的痛点。

许多开发者 / 办公族同时用 JSON 格式化、Base64、UUID、Postman、SSH 客户端、PDF 合并、二维码生成、字帖打印 …… 安装、更新、维护各种小工具的成本很高。**Lele Tools** 把这些常用功能集中到一个 App 里，统一界面、统一快捷键、统一数据存储，专注**做减法**。

## 谁在用

- **开发者**：JSON / YAML / XML 格式化、Base64 / 进制转换、HTTP 客户端、SSH 终端、数据库工具、正则测试
- **运维**：网络扫描、Ping 监控、端口扫描、路由追踪、Whois、Hosts 编辑、SSH/SFTP
- **办公族**：富文本笔记（含 PDF 导出）、PDF 合并/盖章/水印、图片压缩格式转换、二维码、Markdown 转长图
- **个人用户**：工资计算（含节假日数据）、字帖生成、密码生成器、UUID、邮编/手机归属地查询
