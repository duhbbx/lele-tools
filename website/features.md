# 功能一览

按类别分组的 60+ 工具。每个工具都是独立模块，可单独使用。

## 编码 & 格式化

| 工具 | 说明 |
|------|------|
| JSON 格式化 | 美化 / 压缩 / 校验，支持深层嵌套 |
| XML 格式化 | 缩进美化、属性按字母排序 |
| YAML 格式化 | 把杂乱的 YAML 标准化 |
| Base64 编解码 | 文本 / 文件双向 |
| 进制转换 | 2 / 8 / 10 / 16 任意互转 |
| 正则测试 | 实时高亮匹配、捕获组、命名分组 |
| 正则内容生成 | 给定正则反向生成符合的样本 |
| UUID 生成器 | v1 / v4 / v7 任选 |
| 文本加解密 | AES / DES / 3DES / 自定义 key |
| 字符统计 | 字符数 / 词数 / 行数 / 段落数 |
| HTML 特殊字符表 | 备查的实体编码表 |
| HTTP 状态码 | 离线版状态码大全 |

## 网络工具

| 工具 | 说明 |
|------|------|
| HTTP 客户端 | 类 Postman，请求历史 / 集合管理 |
| Ping 工具 | 实时延迟图表 + 抖动统计 |
| 路由追踪 | 跨平台 traceroute，节点地理位置可视化 |
| 端口扫描 | 单机 / 网段批量扫描 + 可 kill 占用进程 |
| 网络扫描 | LAN 内 ARP 扫描所有在线设备 |
| Telnet | 简易 Telnet 客户端 |
| Whois 查询 | 域名所有人信息 |
| IP 查询 | 公网 IP + 离线 ip2region 数据 |
| 路由测试 | 多目标 Tab 化路由质量监控 |
| SSH 客户端 | 内置终端 + SFTP 文件传输 |
| FTP 客户端 | 主动 / 被动模式 |
| FTP 服务器 | 一键起 FTP server 供他人接入 |
