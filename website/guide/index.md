# 使用指南

Lele Tools 是一个工具集合 App。**没有"全局教程"**，每个工具是独立的小应用。

本节针对**最常用的几十个工具**给操作截图和使用要点。左侧目录按类别浏览，搜索框（右上角）可以按关键字查找。

## 主界面布局

```
┌────────────┬──────────────────────────────┐
│            │                              │
│  左侧      │       右侧工具内容区          │
│  工具列表  │                              │
│            │                              │
│  (60+ 项)  │                              │
│            │                              │
└────────────┴──────────────────────────────┘
                       │ 状态栏 / 截图工具      │
                       └──────────────────────┘
```

- **左侧**：工具列表，按"最近使用"自动排序最近用过的放最前。可以拖动调整宽度。
- **右侧**：当前选中工具的界面，跟桌面 App 一样独立可用。
- **顶部**：标签栏（多窗口工具可同时开多个 tab）
- **底部**：状态栏，文本提示 + 截图工具入口

## 通用快捷键

| 快捷键 | 作用 |
|--------|------|
| `Cmd/Ctrl + ,` | 打开设置 |
| `Cmd/Ctrl + W` | 关闭当前 tab |
| `Cmd/Ctrl + T` | 新建 tab（重新打开当前工具） |
| `Cmd/Ctrl + Shift + F` | 在工具列表里搜索 |
| `Cmd/Ctrl + /` | 显示快捷键帮助 |

## 数据存储

每个工具的设置 / 历史记录存在本地 SQLite，路径：

- macOS: `~/Library/Application Support/duhbbx/lele-tools/`
- Windows: `%LOCALAPPDATA%/duhbbx/lele-tools/`
- Linux: `~/.local/share/duhbbx/lele-tools/`

可以备份这个目录迁移到其他机器。
