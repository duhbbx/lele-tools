# 汉字字帖生成器

这是一个功能完整的汉字字帖生成工具，支持多种字帖类型、格子样式和自定义选项。

## 功能特性

### 🎯 字帖类型
- **描字帖**: 生成带有完整汉字的描摹练习字帖
- **笔顺贴**: 显示汉字笔画顺序的练习字帖
- **控笔贴**: 专注于控笔练习的字帖
- **字帖模板**: 生成空白模板供自由练习

### 📐 格子样式
- **无格子**: 纯文字练习
- **方格**: 简单的方形边框
- **田字格**: 传统的四宫格练习格
- **米字格**: 带对角线的八分格
- **米字回宫格**: 米字格加内框
- **回宫格**: 双重边框格

### 🎨 格子颜色
- **绿色**: 护眼的传统绿色格线
- **黑色**: 经典黑色格线
- **红色**: 醒目的红色格线
- **自定义**: 任意颜色选择

### ⚙️ 高级选项
- **显示拼音栏**: 在字符上方添加拼音区域
- **填充拼音**: 自动填入汉字拼音
- **显示中文栏**: 显示汉字书写区域
- **填充中文**: 自动填入汉字内容
- **字体大小调节**: 20-100px 可调
- **格子大小调节**: 50-150px 可调

### 🖨️ 输出功能
- **实时预览**: 即时查看字帖效果
- **打印输出**: 支持标准A4纸张打印
- **预览对话框**: 全屏预览字帖效果

## 使用方法

### 1. 选择字帖类型
在顶部选择您需要的字帖类型：
- 描字帖：适合初学者描摹练习
- 笔顺贴：学习正确的书写笔顺
- 控笔贴：提高书写控制能力
- 字帖模板：自由练习书写

### 2. 输入练习内容
在文本框中输入要练习的汉字，支持任意汉字内容。

### 3. 选择格子样式
根据练习需要选择合适的格子类型：
- 初学者推荐使用田字格或米字格
- 有一定基础可选择方格或无格子
- 特殊练习可选择回宫格等

### 4. 设置格子颜色
选择合适的格子颜色，或使用自定义颜色。

### 5. 配置显示选项
- 勾选"显示拼音栏"可在字符上方显示拼音
- 勾选"填充拼音"会自动添加拼音内容
- 勾选"显示中文栏"和"填充中文"控制汉字显示

### 6. 调整大小设置
- 字体大小：控制汉字的显示大小
- 格子大小：控制练习格的大小

### 7. 生成和打印
- 点击"生成"查看字帖效果
- 点击"预览"全屏查看字帖
- 点击"打印"输出到纸张

## 技术实现

### 核心组件

1. **ChineseCopybook**: 主界面控制器
   - 管理整体界面布局
   - 处理用户交互
   - 协调各个组件

2. **CopybookConfig**: 配置管理
   - 存储所有字帖设置
   - 类型安全的配置选项
   - 默认值管理

3. **绘制引擎**: 字帖生成核心
   - 基于QPainter的高质量绘制
   - 支持多种格子类型
   - 灵活的布局算法

### 格子绘制算法

#### 田字格 (TianZi Grid)
```cpp
void drawTianZiGrid(QPainter& painter, const QRect& rect, const QColor& color) {
    painter.setPen(QPen(color, 2));
    painter.drawRect(rect);

    painter.setPen(QPen(color, 1));
    // 垂直中线
    painter.drawLine(rect.center().x(), rect.top(), rect.center().x(), rect.bottom());
    // 水平中线
    painter.drawLine(rect.left(), rect.center().y(), rect.right(), rect.center().y());
}
```

#### 米字格 (MiZi Grid)
```cpp
void drawMiZiGrid(QPainter& painter, const QRect& rect, const QColor& color) {
    drawTianZiGrid(painter, rect, color);

    painter.setPen(QPen(color, 1));
    // 对角线
    painter.drawLine(rect.topLeft(), rect.bottomRight());
    painter.drawLine(rect.topRight(), rect.bottomLeft());
}
```

## 配置示例

### 基础练习配置
```cpp
CopybookConfig config;
config.type = CopybookType::Tracing;      // 描字帖
config.gridType = GridType::TianZi;       // 田字格
config.gridColor = QColor(0, 128, 0);     // 绿色
config.fontSize = 48;                     // 48px字体
config.showPinyinRow = true;              // 显示拼音
config.fillPinyin = true;                 // 填充拼音
```

### 高级练习配置
```cpp
CopybookConfig config;
config.type = CopybookType::StrokeOrder;  // 笔顺贴
config.gridType = GridType::MiZi;         // 米字格
config.gridColor = Qt::red;               // 红色格线
config.fontSize = 60;                     // 60px字体
config.showChineseRow = true;             // 显示中文行
config.fillChinese = false;               // 不填充中文
```

## 故障排除

### 常见问题

1. **字体显示异常**
   - 检查系统字体库
   - 确认字体文件完整性
   - 尝试使用系统默认字体

2. **打印效果不佳**
   - 调整打印机设置
   - 检查纸张规格设置
   - 确认打印质量选项

3. **拼音显示错误**
   - 确认拼音数据库完整
   - 检查字符编码
   - 验证输入文本格式

4. **生成速度较慢**
   - 减少单页字符数量
   - 降低字体和格子大小
   - 关闭不必要的显示选项

### 性能优化建议

1. 合理设置字符数量，避免单页内容过多
2. 使用标准字体，避免复杂字体文件
3. 适当调整预览质量，提高响应速度
4. 定期清理临时文件和缓存
