# 摄像头工具 (CameraTool)

## 功能概述

摄像头工具提供了完整的摄像头管理和图像捕获功能，包括：

- 📹 **摄像头枚举** - 自动检测和列出系统中可用的摄像头设备
- ▶️ **摄像头控制** - 开启/关闭摄像头，实时预览摄像头流
- 📸 **拍照功能** - 捕获摄像头图像并保存或复制到剪贴板
- 🖥️ **屏幕截图** - 截取指定屏幕的图像
- ⚙️ **灵活设置** - 自定义保存路径、图片质量、分辨率等

## 主要特性

### 摄像头管理
- 自动检测系统中的所有摄像头设备
- 支持多摄像头切换（需要先关闭当前摄像头）
- 实时显示摄像头状态和设备信息
- 支持多种分辨率设置（640x480 到 1920x1080）

### 图像捕获
- **拍照功能**：从摄像头捕获高质量图像
- **屏幕截图**：支持多显示器环境，可选择特定屏幕截图
- **灵活保存**：可选择保存到文件或复制到剪贴板
- **自动命名**：基于时间戳的自动文件命名

### 用户界面
- 直观的分区布局：控制面板 + 预览区域
- 实时预览摄像头画面
- 详细的操作日志记录
- 响应式设计，支持窗口缩放

### 设置选项
- **保存路径**：自定义图片保存目录
- **自动保存**：开启/关闭自动保存到文件功能
- **图片质量**：1-100% 可调节的JPEG压缩质量
- **设置持久化**：所有设置自动保存和恢复

## 技术实现

### 依赖库
- **Qt Multimedia**：摄像头访问和媒体处理
- **Qt MultimediaWidgets**：视频预览控件
- **QCamera**：摄像头设备管理
- **QImageCapture**：图像捕获功能
- **QMediaDevices**：设备枚举

### 核心组件
- `QCamera`：摄像头控制
- `QMediaCaptureSession`：媒体捕获会话管理
- `QVideoWidget`：视频预览显示
- `QImageCapture`：图像捕获处理
- `QScreen`：屏幕截图功能

## 使用说明

### 基本操作流程

1. **启动工具**：在工具列表中选择"摄像头工具"
2. **选择摄像头**：从下拉列表中选择要使用的摄像头设备
3. **开启摄像头**：点击"开启摄像头"按钮，预览区域将显示实时画面
4. **拍照**：点击"拍照"按钮捕获当前画面
5. **截屏**：选择屏幕后点击"截取屏幕"进行屏幕截图

### 高级功能

#### 分辨率调整
- 在摄像头控制区域选择合适的分辨率
- 分辨率更改将在下次启动摄像头时生效

#### 保存设置
- 点击"浏览"按钮选择图片保存目录
- 取消"自动保存到文件"将图片复制到剪贴板
- 调整图片质量滑块设置JPEG压缩质量

#### 多屏幕截图
- 在屏幕截图区域选择要截取的屏幕
- 支持主屏幕和扩展屏幕

## 注意事项

### 系统要求
- 需要安装Qt Multimedia组件
- Windows 10或更高版本推荐
- 需要摄像头设备的访问权限

### 使用限制
- 同时只能启动一个摄像头设备
- 录制功能暂未实现（预留接口）
- 某些摄像头可能不支持所有分辨率

### 故障排除
- **摄像头无法启动**：检查设备是否被其他应用占用
- **预览黑屏**：尝试重新选择摄像头设备或重启应用
- **权限错误**：确保应用有访问摄像头的权限
- **保存失败**：检查保存路径是否存在且有写入权限
