# FTP 客户端工具

这是一个基于 Qt 的 FTP 连接和文件管理工具，提供类似于 FileZilla 的功能。

## 功能特性

### 🔌 FTP 连接管理
- **连接配置**: 保存和管理多个 FTP 连接配置
- **认证方式**: 支持用户名和密码认证
- **被动模式**: 支持被动模式和主动模式连接
- **连接测试**: 在保存前测试连接的有效性
- **连接历史**: 自动保存连接历史和配置

### 📁 FTP 文件管理
- **双面板设计**: 左侧本地文件，右侧远程文件
- **文件操作**: 上传、下载、删除、新建目录
- **目录导航**: 双击进入目录，支持返回上级
- **文件信息**: 显示文件大小、修改时间
- **批量操作**: 支持多文件选择和批量传输

### 🚀 传输功能
- **进度显示**: 实时显示文件传输进度
- **断点续传**: 支持大文件的断点续传（计划中）
- **传输队列**: 支持多文件传输队列（计划中）
- **传输日志**: 详细的传输日志记录

## 使用方法

### 1. 创建 FTP 连接

1. 点击左侧面板的 **"新建"** 按钮
2. 填写连接信息：
   - **连接名称**: 给连接起一个易识别的名称
   - **主机地址**: FTP 服务器的 IP 地址或域名
   - **端口**: FTP 服务端口（默认 21）
   - **用户名**: FTP 登录用户名
   - **密码**: FTP 登录密码
   - **被动模式**: 选择是否使用被动模式
3. 点击 **"测试连接"** 验证配置
4. 点击 **"确定"** 保存连接

### 2. 建立 FTP 连接

- **方式一**: 双击左侧连接列表中的连接
- **方式二**: 选中连接后点击 **"连接"** 按钮
- **方式三**: 右键点击连接选择 **"连接"**

### 3. 使用 FTP 文件管理

连接成功后，可以在 **"FTP文件管理"** 标签页中：

#### 导航操作
- **双击目录**: 进入子目录
- **双击 ".."**: 返回上级目录
- **刷新**: 点击刷新按钮更新文件列表

#### 文件操作
- **上传文件**: 选择本地文件，点击 **"上传 →"**
- **下载文件**: 选择远程文件，点击 **"← 下载"**
- **删除文件**: 选择远程文件，点击 **"删除"**
- **新建目录**: 点击 **"新建目录"** 并输入目录名

## 技术实现

### 依赖库
- **Qt6**: GUI 框架和网络库
- **QTcpSocket**: TCP 连接实现
- **C++20**: 现代 C++ 语言特性

### 核心组件

1. **FTPClient**: 主界面控制器
2. **FTPConnectionManager**: 连接配置管理
3. **FTPConnection**: FTP 连接封装
4. **FTPFileManager**: 文件管理界面
5. **FTPConnectionDialog**: 连接配置对话框

### 协议实现

本工具实现了基本的 FTP 协议命令：
- **USER/PASS**: 用户认证
- **CWD**: 改变目录
- **LIST**: 列出目录内容
- **STOR**: 上传文件
- **RETR**: 下载文件
- **DELE**: 删除文件
- **MKD**: 创建目录
- **RMD**: 删除目录

### 安全考虑

- 连接配置本地存储（计划加密）
- 支持被动模式防火墙穿透
- 传输过程错误处理
- 连接超时管理

## 使用示例

### 连接到服务器
```cpp
FTPConnectionInfo info;
info.name = "我的FTP服务器";
info.hostname = "ftp.example.com";
info.port = 21;
info.username = "user";
info.password = "password";
info.usePassiveMode = true;
```

### 文件操作
```cpp
// 上传文件
fileManager->uploadFile("/local/path/file.txt", "/remote/path/file.txt");

// 下载文件
fileManager->downloadFile("/remote/path/file.txt", "/local/path/file.txt");

// 创建目录
connection->createDirectory("/remote/path/new_directory");
```

## 扩展功能

该工具设计时考虑了扩展性，可以轻松添加：

- **SFTP 支持**: 加密的文件传输协议
- **FTPS 支持**: SSL/TLS 加密的FTP
- **同步功能**: 本地和远程目录同步
- **定时任务**: 自动化文件传输任务
- **传输统计**: 传输速度和统计信息

## 故障排除

### 常见问题

1. **连接失败**
   - 检查主机名、端口是否正确
   - 确认防火墙设置
   - 验证用户名和密码

2. **被动模式问题**
   - 尝试切换被动模式设置
   - 检查服务器被动模式支持
   - 确认防火墙允许数据连接

3. **传输失败**
   - 检查网络连接稳定性
   - 验证文件权限
   - 确认磁盘空间充足

### 调试模式

开发时可以启用详细日志：
```cpp
// 在连接建立时查看FTP响应
connect(connection, &FTPConnection::responseReceived,
        [](const QString& response) {
            qDebug() << "FTP Response:" << response;
        });
```
